# ui_kits/docs — &gt;bitforge docs

Bitforge docs / architecture page on the cream-paper surface.

## Components
- `DocsHeader` — `## eyebrow` + `## h1` heading pattern
- `DocsBody` — body paragraphs with mono type and inline code
- `PaperCard` — bordered card with brutal-shadow + colored category dot eyebrow + title + sub + optional chips
- `ThreeLayerDiagram` — WIRE / VAULT / ANVIL row → FORGE, with dashed connector lines
